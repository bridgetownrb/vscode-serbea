# Serbea syntax highlighting for VS Code

Highlight your Serbea code!

If you're using the VSCode Icons package, add this to your `settings.json`:

```json
  "vsicons.associations.files": [
    {
      "icon": "erb",
      "extensions": ["serb"],
      "format": "svg"
    }
  ]
```
